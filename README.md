# Projeto Clean House

## Iniciar em modo desenvolvedor

```bash
# npm
npm run install
npm run dev

# yarn
yarn install
yarn dev
```

## Iniciar em modo de produção

```bash
# npm
npm run install
npm run build
npm run start

# yarn
npm run install
npm run build
npm run start
```

Acesse o projeto na sua máquina, usando o link [http://localhost:8786](http://localhost:8786).
